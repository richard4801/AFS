# AFS

Apex Fiction Studio — writer, admin, and Senior Editor dashboards.

Dashboards: `dashboard/index.html` (writer), `dashboard/admin.html` (admin), `dashboard/editor.html` (Senior Editor).

Comment system SQL lives in `sql/universal_comments.sql` and `sql/se_edit_access.sql`. Senior Editor chapter recommendations live in `sql/se_chapter_recommendations.sql`. Private admin/Senior Editor chat (reply-to, editing) lives in `sql/private_chat.sql`.

Live at https://apexfictionstudio.com/.
